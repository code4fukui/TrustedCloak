# TrustedCloak

QRコードベースの検証機能を備えた安全なクローク管理システム。

## デモ
https://code4fukui.github.io/TrustedCloak/

## 機能
- 各クロークの預かりに対して一意のQRコードを生成
- デジタル署名を使用してクロークの預かりの真正性を検証
- システムの公開鍵/秘密鍵ペアをリセット
- ブラウザのローカルストレージにクロークの預かり履歴を保存

## 使い方
1. 預かり番号を入力し、「預かり証明書発行」をクリックしてQRコードを生成します。
2. 組み込みのQRコードリーダーを使用してQRコードをスキャンし、預かり情報を検証します。
3. システムの公開鍵が表示され、必要に応じてリセットできます。
